For those nights when you are fighting over whose turn it is to pick, What2Eat makes it simple.

A 1 week project to learn Django REST Framework and gain more Experience with JWT auth and unit/integration testing.

Frontend is built in JavaScript with React and uses Bootstrap, Axios, and React Testing Library with Jest

Backend is built in Python using Django, Django REST Framework, and PostgreSQL
